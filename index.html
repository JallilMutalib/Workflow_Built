03 / n8n
🔥 Hot / Warm / Cold Scoring

Lead Capture → AI Qualification → Auto Follow-up

A Tally form webhook triggers the full pipeline: fields are extracted, OpenAI scores the lead as Hot/Warm/Cold, a Code node parses the score, and the Switch routes to three separate Gmail follow-ups — a calendar booking link for Hot leads, a nurture email for Warm, and a polite no for Cold. Google Sheets logs every result.

Node Flow
🔗 Webhook (Tally) → ✏️ Set Fields → ⚡ Respond to Webhook → 🤖 OpenAI Score Lead → { } Code – Parse Score → 🔀 Switch by Score → 📧 Gmail (Hot/Warm/Cold) → 📊 Log to Sheets

04 / Make.com
🔁 Every 15 min · Multi-route

Google Sheets → Multi-Route Gmail + Notion CRM

Built entirely in Make.com — watches Google Sheets every 15 minutes for new rows, routes each lead through a Router module based on service interest (VA, Social Media, Automation, Web Dev), sends a tailored Gmail, and creates a Notion database entry in the correct category. All four routes run in parallel.

Module Flow
📊 Google Sheets Watch → 🔀 Router (4 routes) → 📧 Gmail Send (×4) → 📋 Notion Create (×4)
